# SmileCloudSolution


## צורת הקבצים
- **InputForm.jsx**: קבלת שלוש נקודות מהמשתמש.
- **TriangleCanvas.jsx**: ציור המשולש וחישוב הזוויות באמצעות חוק הקוסינוסים.
- **math.js**: פונקציות חישוב מתמטיות.
- **App.jsx**: ניהול ה־state ומעבר בין מסך הקלט למסך התצוגה.
